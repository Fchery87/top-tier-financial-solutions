# Issue Tracker

Issues for this repo are tracked in GitHub Issues for `Fchery87/top-tier-financial-solutions`.

## Consumer Rules

- Use GitHub Issues as the source of truth for planned work, bugs, PRDs, and triage state.
- Use `gh issue` commands when creating, reading, updating, or labeling issues.
- Target the repository `Fchery87/top-tier-financial-solutions` explicitly when the GitHub CLI cannot infer it.
- Do not create local `.scratch/` issue files unless the user explicitly asks to switch to a local-markdown workflow.
- Keep issue bodies self-contained enough for an AFK agent to pick up without relying on private chat context.
